# Cursor Project Starter

This repository is a generic starter template for projects using **Cursor AI**
with a **doc-first, Amazon-style workflow**.

## What this repo provides
- Cursor rules for consistent behavior (`.cursor/rules`)
- A doc-first feature workflow (PRFAQ, design, tasks)
- Reusable templates for features, decisions, and PRs
- A scalable structure suitable for web apps, APIs, or internal tools

## Core principles
- Write before building
- One feature = one owner
- Small, reversible changes
- Decisions are documented
- Docs are part of the Definition of Done
